Manager * Manager::defaultManager ( )
static

returns the defaultManager, just in case you don't need separated managers. To free the memory of the default manager call destroyDefaultManager(), the memory is NOT automatically freed at the end of the program.

void Manager::destroyDefaultManager ( )
static

removes the default manager, and resets its memory. It is safe to call defaultManager before destruction and a new instance will be created.
